Project Concept

The goal of this project is to create an interactive display onto thin air.

Use Cases

  • Use air as a bigger display to make the content easier to see.
  • Will provide users with a 3D-esque image. This will provide a better user experience.
  • It has a lot of potential in the advertising industry. People can see life-size 3D-esque images of models.
  • Can be used as an alternative to large-size display screens. It makes the user experience more interactive and engaging.

Competitors

Displair

  • Allows human interaction with a touchscreen out of thin air.
  • Still in development, and not yet shipped.

FogScreen

  • Instead of shooting out compressed air, simulates fog for projecting the screen onto.
  • Does not detect human gestures of any sort.
  • Currently shipped; Can buy/rent this product.

HelioDisplay

  • Allows large size projections that can be used under normal lighting conditions
  • Does not detect human gestures of any sort.

Coachella 2013

  • For Tupac's performance, used a transparent screen to project onto; not truly a walk-through kind of environment
  • Did a variation of Pepper's ghost, which will not work for close ups.

Microsoft's Vermeer

  • More similar to a 360 degree hologram
  • Tracks user's interaction with the product
